# artl-torch

Outlier-robust neural network training via the **Augmented and Regularized Trimmed Loss (ARTL)**.

---

## Installation

```bash
pip install git+https://github.com/oknakfm/artl-torch.git
```
**Requirements:** Python ≥ 3.9, PyTorch ≥ 1.9, NumPy ≥ 1.20.

```bash
python demo/run_demo.py
```
